# Inventario de la Tienda Lolita

## Introducción

Este inventario está dirigida para un tienda grande donde se llevará acabo todas las ejecuciones que debe tener un inventario para que esten organizadas de la mejor manera.

## Tecnologías utilizadas

- Python
- Git graph
- Conventional commits

## Intrucciones de ejecución 

1. Primer debe ir a la terminal.
2. escribir python3 main.py
3. Ejecutar o dar enter.
